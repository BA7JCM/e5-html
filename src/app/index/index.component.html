说明

  1. 此程序部署在我的服务器上，每隔2小时会自动调用outlook api 获取邮件列表
  2. 对于信息: 只会保存必要的 client_id、client_secret，其他任何内容都不会保存，
  3. 会读取授权的outlook账号邮箱邮件，但不会保存任何信息，仅仅是调用api。
  4. 请单独创建一个同域 E5 子账号进行授权，不要使用此账号进行发送、接收个人邮件，以免发生误会
